# Besa Foundation Website

## Project Overview

This is the official website for Besa Foundation - a non-profit organization supporting the BesaChain ecosystem.

## Tech Stack

- Next.js 14 with App Router
- TypeScript
- Tailwind CSS
- shadcn/ui components
- Recharts for data visualization

## Design System

### Colors
- Primary: `#1e3a5f` (navy-800)
- Accent: `#00d4ff` (cyan-400)
- Background: White with navy-50 sections

### Typography
- Font: Inter (Google Fonts)

## Sections

1. Hero - Main landing with value proposition
2. Mission - Foundation pillars and vision
3. Governance - Tokenomics and voting
4. Grants - Funding program details
5. Research - Whitepapers and partnerships
6. Team - Core team and advisors
7. Financials - Treasury transparency
8. News - Updates and press releases
9. Contact - Form and office locations
10. Footer - Links and social media

## Build Commands

```bash
npm install    # Install dependencies
npm run dev    # Development server
npm run build  # Production build (outputs to dist/)
```

## Deployment

Static export configured in next.config.js:
- Output directory: `dist/`
- Format: Static HTML
